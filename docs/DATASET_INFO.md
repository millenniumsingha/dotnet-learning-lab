# Deep Moor Environmental Data

This dataset contains environmental observations from the Deep Moor station.

## Files
- `Environmental_Data_Deep_Moor_2012.txt`
- `Environmental_Data_Deep_Moor_2013.txt`
- `Environmental_Data_Deep_Moor_2014.txt`
- `Environmental_Data_Deep_Moor_2015.txt`

## Format
Text-based, tab/space separated.

## Columns
1. **Date** (YYYY-MM-DD)
2. **Time** (HH:mm:ss)
3. **Barometric_Pressure**
4. **Other metrics** (varies by file)

## Usage
These files are used by the **WeatherTrend** WPF application to visualize barometric pressure trends over time.
